Создание пользовательских свойств

Пользовательские свойства сгруппированы под названием Классификации пользовательских свойств. Классификации и связанные с ними свойства назначаются точкам на чертеже путем использования групп точек. После добавления пользовательских свойств к группе точек можно добавить значения для каждого свойства путем редактирования отдельных точек или путем импорта значений из внешнего файла.

Для отображения информации о пользовательском свойстве точке необходимо создать соответствующий стиль метки точки.

В следующем упражнении вы создадите пользовательскую классификацию свойств группы точек.

  1. Из папки Учебные файлы/Точки откройте файл скважины.dwg. Данный файл содержит точки, обозначающие местоположения геологических скважин.
  2. На вкладке Навигатор щелкните кнопкой мыши по коллекции Точки, из контекстного меню выберите команду Редактировать точки. Просмотрите данные точек, обратите внимание, что все точки имеют описание СКВ.
  3. В коллекции Группы точек создайте новую группу точек под названием Скважины.
  4. В свойствах группы точек Скважины задайте стиль точек Геологическая скважина, стиль меток точек Отметка и Описание.
  5. В качестве условия включения точек в группу укажите совпадение исходного описания СКВ. В результате все точки чертежа должны быть включены в группу Скважины и отображаться в соответствии с указанными стилями. Во вкладке «Группы точек» поставить галочку «Все точки» Создайте новый стиль меток, в которых бы отображался номер скважины. При этом следует иметь в виду, что номер скважины совпадает с номером точки. Для создания такого стиля выполните следующие действия:
  6. На основе стиля меток Отметка и Описание создайте новый стиль меток Данные скважины.
  7. В окне Создание стиля метки на вкладке Компоновка в списке Имя компонента выберите компонент Описание точки.
  8. Чтобы задать значение свойства Содержимое активизируйте Редактор компонентов текста. Добавьте к свойству Исходное описание свойство Номер точки, разделив их знаком дефис. Нажмите ОК.
  9. Для компонента Отметка точки задайте значение свойства «Смещение по Х» равным 0.00мм
  10. Для группы точек Скважины задайте стиль меток точек Данные скважины. Убедитесь, что метка точки отображается согласно заданному стилю
  11. Сравните полученный результат с файлом скважины_1.dwg, расположенным в папке Учебные файлы/Точки/Результаты.

Создайте пользовательскую классификацию свойств для группы точек Скважины.

Для этого выполните следующие действия.

  1. На вкладке Параметры разверните коллекцию Точка. Нажмите правую кнопку мыши на элементе Классификации пользовательских свойств. Выберите команду Создать.
  2. В диалоговом окне Классификации пользовательских свойств введите имя классификации Свойства скважины. Нажмите ОК. Создается новая классификация, она добавляется к списку классификаций пользовательских свойств.
  3. На вкладке Параметры разверните коллекцию Классификации пользовательских свойств. Нажмите правую кнопку мыши на имени Свойства скважины. Выберите команду Создать.
  4. В диалоговом окне Новое пользовательское свойство укажите значения согласно рис.
  5. Нажмите ОК. Это свойство добавляется в список свойств Свойства скважины.
  6. Добавьте еще два свойства к классификации Свойства скважины с использованием значений, приведенных в таблице.
    Имя Описание Тип Нижнее граничное значение Верхнее граничное значение Значение по умолчанию
    Установление воды Глубина установления воды в метрах Расстояние 0.0м 2.0м 0.0м
    Дата Дата проходки Строковый - - 01.01.2009
  7. Щелкните кнопкой мыши по имени классификации Свойства скважины, убедитесь, что на виде списка отображаются все три созданные пользовательские свойства.
  8. Сохраните файл.
