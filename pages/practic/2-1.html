Работа с чертежами

Чтобы создать новый чертеж

Новый чертеж создается при помощи стандартных команд AutoCAD.

  1. Нажмите кнопку -> Создать.
  2. В диалоговом окне "Выбор шаблона" выберите шаблон, на основе которого будет создан чертеж, и нажмите "Открыть".

Чтобы открыть существующий чертеж:

Чтобы открыть существующий чертеж, воспользуйтесь стандартными командами AutoCAD.

Если открыто несколько чертежей, имя активного чертежа отображается в дереве"Навигатор" жирным шрифтом.

Прим.: Для смены активного чертежа нажмите правую кнопку мыши на имени чертежа в главном представлении дерева "Навигатор", затем выберите "Перейти".

  1. Нажмите кнопку -> Создать.
  2. В диалоговом окне "Выбрать файл" откройте содержащую чертеж папку, выберите чертеж и нажмите "Открыть".

Доступ к файлам примеров

Файлы примеров доступны в меню приложения.

В меню приложения выберите команду "Открыть" -> "Файлы примеров", а затем выберите одну из следующих команд:

  • Установленные файлы примеров: отображение папки "Файлы примеров" в диалоговом окне "Выбор файла".
  • Интерактивные файлы примеров: открытие интерактивной веб-страницы файлов примеров для AutoCAD.

Использование шаблонов чертежа

  1. На вкладке "Навигатор" окна "Область инструментов" выберите "Главное представление" из раскрывающегося списка.
  2. Разверните коллекцию "Шаблоны чертежа" и щелкните правой кнопкой мыши на шаблоне:
    • Создать новый чертеж на основе шаблона.
    • Открыть шаблон.

Создание файла шаблона чертежа из существующего чертежа

  1. Нажмите кнопку -> Создать.
  2. В диалоговом окне "Выбор файла" выберите файл, который требуется использовать в качестве шаблона.
  3. Нажмите "OK".
  4. Нажмите кнопку -> Сохранить как.

  5. В диалоговом окне "Сохранение чертежа" в поле "Тип файла" выберите тип файла для шаблона чертежа.

    DWT-файлы должны сохраняться в формате файла чертежа текущей версии программы. Чтобы создать файл DWT в формате предыдущей версии, сохраните данный файл в требуемом формате DWG, а затем переименуйте файл DWG, используя расширение DWT.

  6. В поле "Имя файла" введите имя шаблона.
  7. Нажмите "Сохранить".
  8. Введите описание шаблона.
  9. Нажмите "OK".
