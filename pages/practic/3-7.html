Импорт точек из текстового файла

В данном упражнении для формирования топоплана используются точки съемки, хранящиеся в текстовом файле. Для удобства работы все имеющиеся точки будут распределены по группам. Каждой группе будут назначены свои стили и, таким образом, автоматически вставлены в чертеж условные обозначения.

  1. Создайте новый чертеж.

  2. Из папки “Учебные файлы/Точки” откройте файл “точки.txt”. В данном файле содержатся данные точек в форме:

    Номер точки, Восток, Север, Отметка, Описание.

    Можете закрыть файл.
  3. В окне Область инструментов на вкладке Навигатор нажмите правую кнопку мыши на элементе Точки. Выберите команду Создать.
  4. В диалоговом окне Создание точек щелкните по кнопке (Импорт точек).
  5. В открывшемся окне Импорт точек в разделе «выбранные файлы» нажмите на кнопку и выберите файл “точки.txt” Из папки “Учебные файлы/Точки”.
  6. Затем, в разделе «Укажите формат файла точек» выберите «PENZD (разделение пробелами)». В предварительном просмотре будут видны названия столбцов. Они соответствую нужным нам (см. п. 2).
  7. Нажмите «ОК»
  8. Следующим шагом нажмите ПКМ на пункт «Точки» вкладки «Навигатор» и выберите «Показать». Программа покажет импортированные точки.

  9. Затем нажмите ПКМ на пункт «Группы точек» панели «Навигатор» и выберите пункт «создать». Заполните следующие вкладки в соответствии с изображениями:

    Вкладка «информация»:

    Вкладка «Включить»:

    Затем нажать кнопку «Применить», затем «ОК»

  10. Перейдите на вкладку «Группы точек» панели «Навигатор». Если параметр был создан правильно, то на вкладке отобразится список точек, имеющих исходное описание Земля.
  11. Аналогичным образом создать еще 4 группы точек в соответствии с таблицей:

    Наименование группы Стиль точек Стиль меток Исходное описание
    Скважины Drill Hole Elevation and Description Скв*
    Столбы Guy Pole Description Столб*
    Кусты Shrub - 1.5m - Куст
    Здание Стандартный Point Number Only Здание
