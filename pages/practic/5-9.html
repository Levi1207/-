Создание и редактирование трасс для смещения

Создание трасс для смещения.

Создание смещений осевой линии трассы.

  1. Из папки Учебные файлы/Трассы/8.Создание трасс для смещения откройте файл Align-6A.dwg. Этот чертеж содержит поверхность и трассу.
  2. Выберите команду вкладки "Главная" - панели "Создать проектные данные" - раскрывающегося списка "Трасса" - "Создать трассу для смещения".
  3. Выберите трассу на чертеже.
  4. В диалоговом окне "Создать трассы для смещения" задайте следующие значения параметров: Можно также задать критерии проектирования трасс для смещения.
  5. Нажмите "OK". С каждой стороны от осевой линии трассы создается трасса для смещения. В окне "Область инструментов", на вкладке "Навигатор" разверните коллекцию "Трассы"-"Трасса для смещения". Обратите внимание на то, что две созданные трассы помещены в коллекцию "Трассы для смещения". На основе трасс для смещения можно создавать профили, виды профилей и группы осей сечения, точно так же как для стандартных трасс типа осевая линия.

    6. Редактирование трассы для смещения.

    Возможность применения инструментов редактирования геометрической формы к трассам для смещения зависит от того, является ли данная трасса статической или динамической. Если трасса для смещения сохраняет динамическую связь с родительской осевой линией трассы, то редактирование геометрии трассы для смещения невозможно. В случае статической трассы для смещения можно пользоваться инструментами на панели "Инструменты компоновки трассы".

    В диалоговом окне "Свойства трассы" предусмотрена вкладка "Параметры смещения". На ней можно изменять такие параметры, как номинальное значение смещения или начальный и конечный пикеты.

    Изучение геометрических свойств трассы для смещения.

  6. Выберите левую трассу для смещения.
  7. Выберите команду вкладки "Трасса" - панели "Редактирование" - "Редактор геометрии".

    8. Редактирование параметров трассы для смещения.

  8. Выберите команду вкладки "Трасса" - панели " Редактирование " - раскрывающегося меню "Свойства трассы" - "Свойства трассы"
  9. В диалоговом окне "Свойства трассы" на вкладке "Информация" разверните список "Тип".
  10. Нажмите Esc.
  11. Перейдите на вкладку "Параметры смещения".
    Эта вкладка служит для уточнения проектных данных трассы для смещения. Если требуется, чтобы внесение изменений в геометрию родительской трассы не влияло на трассу для смещения, с помощью списка "Режим обновления" преобразуйте трассу в статическую.
  12. Задайте следующие значения параметров:
    Номинальное значение смещения: -24,0000
    Конечный пикет: 10+00
  13. Нажмите "OK".
    Теперь трасса для смещения оканчивается на пикете 10+00 и смещена на расстояние, в два раза большее, чем трасса для смещения на противоположной стороне от осевой линии.

    14. Добавление уширения к трассе для смещения.

  14. Выберите трассу для смещения с правой стороны от осевой линии трассы.
  15. Выберите инструмент вкладки "Трасса" - панели "Редактирование" - "Добавить уширение"
  16. По запросам в командной строке задайте следующие значения параметров.
    Создать участок уширения как новую трассу?: Нет
    Начальный пикет: 150
    Конечный пикет: 1000
    Смещение уширения: 24
    Создается область уширения, и введенные параметры отображаются в диалоговом окне "Параметры трассы для смещения".

    17. Добавление области уширения к области уширения.

  17. Выберите трассу для смещения, которая расположена с правой стороны от осевой линии трассы. Щелкните правой кнопкой мыши. Выберите "Параметры смещения". Можно также добавить область уширения с помощью ручки. Параметры трассы для смещения, в том числе параметры существующей области уширения, отображаются в диалоговом окне "Параметры трассы для смещения".
  18. В диалоговом окне "Параметры смещения" выберите "Добавить уширение".
  19. По запросам в командной строке задайте следующие значения параметров:
    Начальный пикет: 550
    Конечный пикет: 750
    Смещение уширения: 42
  20. Нажмите клавишу ESC для отмены выбора трассы для смещения. В пределах исходной области уширения создается вторая область уширения. Параметры новой области уширения отображаются в диалоговом окне "Параметры трассы для смещения".
