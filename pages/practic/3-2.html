Создание ключей-описателей

Из папки Учебные файлы/Точки откройте файл точки рельефа.dwg. Данный файл содержит векторный вариант съёмки рельефа.

В окне Область инструментов на вкладке Параметры разверните коллекцию Точка. Нажмите правую кнопку мыши на элементе Наборы ключей-описателей, выберите команду Создать.

В диалоговом окне Набор ключей-описателей в поле Имя введите содержательное имя для нового набора, например, Ключи точек рельефа. А в поле Описание введите короткое описание, например Точки существующей поверхности. Нажмите ОК. Создается новый ключ-описатель.

На вкладке Параметры разверните коллекцию Наборы ключей-описателей. Нажмите правую кнопку мыши на элементе Ключи точек рельефа и выберите команду Редактировать ключи.

В открывшемся окне Панорама отобразится Редактор ключей-описателей. В Редакторе ключей-описателей впоследствии будут введены коды исходных описаний и определено, как AutoCAD Civil 3D обработает новые точки с данными кодами. Все записи в столбце Код Редактора ключей-описателей учитывают регистр клавиатуры.

На вкладке Редактор ключей-описателей двойным нажатием кнопки мыши выделите ячейку в столбце Код и измените ее на REL*. Звездочка является групповым символом, поэтому все импортируемые точки, код описания которых начинается строкой REL, после которой следуют любые другие символы, будут обрабатываться в соответствии с параметрами этой строки таблицы.

В столбцах Стиль и Стиль метки точки снимите флажки активности этих параметров. Снятие данных флажков позволит управлять этими параметрами с помощью свойств группы точек.

В столбце Формат введите полное описание в метке точки RELIEF. Затем установите флажок в столбце Слой.

Дважды щелкните по пустому пространству ячейки «Слой» для открытия диалогового окна Выбор слоя. В диалоговом окне Выбор слоя выберите V-NODE-STRM. Нажмите ОК. Данный параметр означает, что точки REL будут обращаться за атрибутами отображения к слою V-NODE-STRM.

Нажмите галочку ( ), чтобы сохранить ключи-описатели и закрыть редактор.

Сохраните файл.
