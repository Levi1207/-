Преобразование точек AutoCAD в точки AutoCAD Civil 3D

В файле точки рельефа.dwg в окне Область инструментов на вкладке Навигатор щелкните правой кнопкой мыши на коллекции Точки. Выберите команду Создать. Перед вами отобразится панель инструментов Создание точек. Команды данной панели сгруппированы в меню по способам создания точек.

На панели инструментов Создание точек нажмите кнопку ( ), чтобы полностью развернуть панель.

В категории Создание точек в полях Запрашивать отметки, Запрашивать имена точек и Запрашивать описания выберите опцию Автоматически. Это означает, что при создании точек имена и описания будут присваиваться по умолчанию. В ходе выполнения преобразования отметка точке AutoCAD Civil 3D будет назначаться в соответствии с отметкой точки AutoCAD.

В поле Описание по умолчанию по умолчанию введите код ключа-описателя REL. В ходе выполнения команды создания точек им будет автоматически присваиваться указанное описание.

Не закрывая панели инструментов Создание точек, отключите видимость слоев так, чтобы на чертеже были видны только точки, расположенные на слое Точки-рельеф.

Для этого необходимо нажать на свойства слоя в ленте инструментов.

В открывшемся окне выделить все слои, кроме точки-рельеф. Для этого надо выделить любой слой и нажать Ctrl+A (выделить все), затем щелкнуть по слою точки-рельеф с зажатым Ctrl. Этот пункт исключится из списка. После чего нажать на иконку активности слоя (лампочки) на любом из слоев:

На панели инструментов Создание точек разверните список команд, объединенных в группу Разное, и выберите пункт Преобразовать точки AutoCAD. Выделите все точки на чертеже. Выделение происходит путем, схожим с инструментом лассо в Adobe Photoshop. После преобразования найденных точек система выдаст запрос о следующих точках AutoCAD, нажмите клавишу Enter для завершения команды. Закройте окно «создание точек»

Включите видимость слоев. Убедитесь, что около имени группы точек Точки рельефа коллекции Точки отобразился значок, предупреждающий о необходимости обновления группы точек. Нажмите ПКМ и выберите Актуализировать.

Отключите видимость слоя Отметки. В коллекции Группы точек щелкните правой кнопкой мыши по названию группы точек Точки рельефа и выберите команду Свойства. В диалоговом окне Свойства группы точек смените стиль отображения точек (например, Дерево широколиственное). Нажмите ОК. Отображение точек изменится в соответствии с указанным стилем.

Полученный результат можете сравнить с файлом точки рельефа_1.dwg, расположенным в папке Учебные файлы/Точки/Результаты.

Для управления цветом обозначения точки и ее метки используйте Диспетчер слоев. Цвет слоя V-NODE-STRM определяет цвет обозначения точки, цвет слоя V-NODE-TEXT – цвет метки точки. Измените цвета обозначения и метки точки по своему усмотрению. В коллекции Точки нажмите правую кнопку мыши на группе точек Точки рельефа. Выберите команду Редактировать точки. Точки отображаются в таблице Редактор точек. Просмотрите и измените их атрибуты. Закройте Редактор точек.
