Изменение стиля профиля

В данном упражнении меняется стиль профиля двумя различными способами. Сначала глобально меняется стиль профиля, что изменяет внешний вид профиля на всех видах профиля. Затем будет показан порядок переопределения стиля профиля на одном виде профиля. В итоге скрываются смещенные профили и их метки.

  1. Продолжим работу с файлом из предыдущего упражнения. В файле npoфиль.dwg выделите созданный нами вид профиля, а затем щелкните ПКМ по сетке вида профиля Гребень дороги со смещениями. Из контекстного меню выберите команду Свойства вида профиля.
  2. В диалоговом окне Свойства вида профиля на вкладке Информация в списке Стиль объекта выберите стиль вида профиля Усеченная сетка. Нажмите кнопку Применить, обратите внимание, как изменился вид профиля.
  3. Перейдите на вкладку Профили.
  4. В таблице в столбце Стиль дважды щелкните на ячейке для профиля Смещение влево, измените стиль профиля, выбрав стиль Профиль поверхности. Нажмите ОК
  5. Нажмите ОК для закрытия окна Свойства вида профиля. Смещенный влево профиль имеет такой же цвет, как профиль осевой дороги.

На следующих этапах будет продемонстрирован другой способ изменения стиля профиля.

  1. На виде профиля Гребень дороги со смещениями щелкните правой кнопкой мыши по профилю Смещение вправо, предварительно выделив его, затем из контекстного меню выберите Свойства профиля.
  2. На вкладке Информация можно изменить свойства линии профиля после того как он построен на виде профиля.
    Измените стиль профиля на Проектный - один цвет. Нажмите ОК.
  3. Сравните результат с файлом профиль_2.dwg, расположенным в папке Учебные файлы/Профили/Результаты.
  4. Сохраните файл.
